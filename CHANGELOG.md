# Changelog

All notable changes to VaporUI will be documented in this file.

## [1.0.0] - 2024-12-26

### Added
- Initial release
- Window component with draggable functionality
- Tab system with smooth transitions
- Button component with ripple effect
- Toggle switch with smooth animations
- Slider component with real-time value updates
- Dropdown menu with smooth open/close
- Text input with focus effects
- Label component with auto-sizing
- Section divider with gradient lines
- Notification system with multiple types (Info, Success, Warning, Error)
- Vaporwave tech dark theme with cyan/pink gradient
- Smooth animations and transitions
- Glow effects on UI elements
- Modular architecture
- Single file build system
- Zero dependencies

### Theme
- Deep dark background (RGB 15, 15, 25)
- Primary purple (RGB 138, 43, 226)
- Accent cyan (RGB 0, 255, 255)
- Accent pink (RGB 255, 20, 147)
- Smooth gradients throughout UI
- Neon glow effects
